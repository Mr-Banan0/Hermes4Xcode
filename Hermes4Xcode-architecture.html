Hermes4Xcode — 基于 Hermes Agent 的优秀 Coding Agent 架构

三层架构：Xcode IDE → Hermes4Xcode (macOS GUI) → Hermes Agent (Backend 引擎)

设计原则

  • • 复用 Hermes 引擎 — 不重复造轮子， Gateway 已经在跑 Agent 循环
  • • GUI 增强 — 所有优化都是给 Hermes 已有的能力加可视化外壳
  • • 可见性优先 — 让用户看到 agent 在做什么（tool calls、构建过程）
  • • 控制感优先 — diff 预览、apply/reject、undo 机制

现有能力（Hermes 已提供）

  • • Tool Execution Engine（ReAct 循环）
  • • 20+ toolsets（file / terminal / MCP / git）
  • • Session 持久化（SQLite）
  • • Skills + Memory 跨会话知识
  • • Cron 定时任务调度
  • • MCP 服务器集成（xcodebuildmcp）

优化路线图

  • Phase 1 (现在)
  • ┝ ToolCall Visualizer — Gateway 工具调用过程可视化
  • ┝ DiffPreviewView — 代码修改预览 + Apply/Reject
  • ┕ BuildLogView 增强 — 构建结果智能解析
  • Phase 2
  • ┝ ConversationStore — 对话持久化 (Codable)
  • ┝ AgentCollaboration — 转发/路由/协作
  • Phase 3
  • ┝ AgentStatusBar — 多 Agent 状态仪表盘
  • ┕ LSP 深度集成 — 符号级代码理解
